# Основная идея проекта:
Мы посылаем запрос через открытый API на сайте Nasa и получаем JSON содержащий в себе картинку дня, описание к ней и название. 

Приложение написано на языке SwiftUI

Стек технологий:
- JSON UrlSession 
- Animations
- Gestures 
- MVVM
- SOLID principles
